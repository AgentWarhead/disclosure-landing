# LESSONS.md — Disclosure (Landing)
> ⚠️ Read BEFORE touching this project. These cost us real time and money.

1. Build 62 gospel applies here too — React 19.1.4 + RN 0.81.6 stack.
2. Single-file HTML/JS/CSS architecture for landing page — keep it simple.
3. First Contact secret sequence: Q1=3, Q2=1, Q3=4, Q4=2.
4. 4 Archetypes: Sentinel (Red), Diplomat (Green), Scholar (Blue), First Contact (Gold).

11. Urgency Bridge section goes between Rules and Product Showcase.
12. Digital ID card must render on a phone screen mockup in Preview.
13. `style={{display:"none"}}` is unreliable — remove nodes instead.
14. Android local builds eat 16GB — check disk space first.
15. Explicit boss approval required for all public deployments.
